RxJava最佳实践

1 用RxJava实现倒计时

效果图

简介

这个实践是我认为比较简单的RxJava的实践,主要用到了RxJava的Operator有

  • interval: 用来实现按一定的间隔发送数据
  • take: 需要计时多久
  • map 开始倒计时

    思路分析

    要用到RxJava实现Android发送验证码倒计时的功能.笔者查询了RxJava的操作符,发现interval正好符合这个功能.
    在点击按钮的时候开始倒计时,将按钮变成不可点击状态,等待倒计时完成后,才可以变成点击状态.

    interval操作符介绍

    create an Observable that emits a sequence of integers spaced by a given time interval

意思是:创建一个在给定事件间隔的有序的interger类型的Observable;
用法:

  • Javadoc: interval(long,long,TimeUnit)//参数:(开始数,间隔数,时间单位);

  • Javadoc: interval(long,long,TimeUnit,Scheduler);//参数:(,,,指定线程);默认在 计算线程中;

    代码

    具体实现代码如下: 

    Observable.interval(0,1,TimeUnit.SECONDS)
            .take(11)//这里我只需要取用11个数据
            .map(new Func1<Long, Long>() {
                @Override
                public Long call(Long aLong) {
                    return 20 - aLong;//模拟从20开始倒计时
                }
            })
            .doOnSubscribe(new Action0() {//subscribe之后，事件发送前执行。
                @Override
                public void call() {
                    mBtnRxjavaCountdown.setEnabled(false);
                }
            })
            .observeOn(AndroidSchedulers.mainThread())//一定要有,因为界面改变需要在UI线程中完成,而默认interval在计算线程中
            .subscribe(new Observer<Long>() {
                @Override
                public void onCompleted() {
                    Toast.makeText(RxJavaActivity.this, "完成", Toast.LENGTH_SHORT)
                            .show();
                    mBtnRxjavaCountdown.setEnabled(true);
                }
                @Override
                public void onError(Throwable e) {
                }

                @Override
                public void onNext(Long aLong) {
                    mTvRxjavaCountDown.setText("" + aLong);
                }
            });

    对此一个倒计时的功能就完成了
