Linux平台搭建Android开发环境

Posted on 2017-07-17 | In Android , NDK | Visitors

简介

最近笔者在学习NDK的开发,用到了Linux.那么Linux下的开发环境搭建也是必不可少的了,这里做个笔记记录;

工具

VMware,Ubuntu,JDK,AndroidStudio

搭建Java环境

Java的环境很简单,这里的JDK在网上流传有2种,一种是Open JDK,另一种是Oracle JDK.
区别:
Open JDK是Java开发工具包的开源实现
Oracle JDKJava开发工具包的官方Oracle版本
我们这里开发Android App,采用Oracle JDK,以避免UI/性问题.

检查环境

终端中输入:java -version
出现: The program ‘java’ can be found in the following packages
证明需要我们安装Java开发环境;

下载Oracle JDK

登录Oracle官网后,根据Ubuntu环境选择最新的JDK包(选择后缀带有tar.gz的),指定保存文件夹,然后下载.具体下载过程就不详述了,这个都会;

安装Java

解压软件包

采用Root用户执行以下命令;
采用Linux命令到指定下载的文件夹.笔者这里是/home/loren/Downloads,将这个文件拷贝到我们需要安装的目录/usr/java/;

cp jdk_181.tar.gz /usr/java

稍等一会,然后进入拷贝好的文件夹,解压:

tar zxvf jdk_181.tar.gz

有可能会出现权限拒绝的情况,如果出现了,执行以下命令后,然后再执行上面的解压命令;

sudo su //切换到root用户

配置环境变量

终端输入vim ~/.bashrc ,最后一行写入以下配置:

JAVA_HOME=/usr/java/jdk1.8.0_101
JRE_HOME=$JAVA_HOME/jre
JAVA_BIN=$JAVA_HOME/bin
CLASSPATH=.:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar:$JRE_HOME/lib
PATH=$PATH:$JAVA_HOME/bin:$JRE_HOME/bin
export JAVA_HOME JRE_HOME PATH CLASSPATH

其中,JAVA_HOME后面的路径是你jdk解压后的路径.
保存(vim 的保存是:按下Esc处于命令行状态,输入::wq即可保存并退出)
Vim几种保存的区别:
:q 直接退出,:q! 强制退出, :wq 保存并退出,:!wq 强制保存并退出
为了让我们的修改立即生效,终端输入执行 source ~/.bashrc
然后在终端输入 java -version 出现了java的版本,证明Java环境配置成功;

loren@ubuntu:~$ java -version
java version "1.8.0_141"
Java(TM) SE Runtime Environment (build 1.8.0_141-b15)
Java HotSpot(TM) 64-Bit Server VM (build 25.141-b15, mixed mode)
loren@ubuntu:~$

搭建Android环境

下载Android Studio

虽然说Ubuntu可以使用终端来安装AS,但是通过源安装的AS版本有点老旧,所以这里我们从官网上下载(请自备梯子),这里我们根据提示来下载并保存就好;

安装Android Studio

拷贝下载的文件(通过火狐浏览器下载的文件在:/home/用户名/Downloads中)到指定的目录,笔者这里是/home/loren/as,打开终端,然后采用解压我们的安装包.

uzip android-studio-ide-162.4069837-linux.zip

解压完成后,进入我们解压后的目录/home/loren/as/android-studio/bin,然后输入命令./studio.sh就可以打开我们的AndroidStudio界面了.

HelloWorld

继续上面的操作,我们打开后是常见项目选择界面:

然后我们进入SDK Manager下载相应的SDK(自备梯子):

下载后,我们创建一个HelloWorld项目,选好指定的品牌,然后一路Next下去就可以了;
然后进入就是熟悉的代码界面,在配置好gradle(重要一点就是更改compileSdkVersion和buildToolsVersion)中的编译版本后,接上手机调试,手机屏幕会出现HelloWorld,证明我们的环境搭建成功了;

创建Studio的快捷方式

创建Studio.desktop文件

在命令行中输入:

sudo vim /usr/share/applications/Studio.desktop

如果我们的Linux没有安装vim编辑器的话,那么有可能会出现找到不vim命令的错误,不过不用担心,只要根据接下来的提示安装vim即可.

编辑Studio.desktop文件

在打开的文件中输入以下内容:

[Desktop Entry]
Name = AS
Comment= android studio
Exec=/home/loren/as/android-studio/bin/studio.sh
Icon=/home/loren/as/android-studio/bin/studio.png
Terminal=false
Type=Application

注意:
1.每一行的结尾处都不能有空格(笔者被坑了好久);
2.Exec后面的路径根据自己studio.sh路径做适当调整.Icon同样如此;
配置完毕后,按 Esc后,再输入:!wq 保存并退出 vim;

添加快捷方式

点击左上角的搜索,输入studio,然后将出现的图标拖到linux左边快捷菜单条中即可.或者输入

nautilus /usr/share/applications/

找到Studio对应的图标,拖到桌面中,也是可以的;

搭建NDK开发环境

既然是在Linux上做开发,那么肯定会需要NDK的场景,随手把NDK环境给配置好吧

下载NDk

下载地址:官网下载(自备梯子),注意区分系统的版本,别下错了;

安装

将下载好的NDK包复制到我们需要的目录中,如笔者这里是todo,然后解压文件todo.解压完成后,用vim打开~/.bashrc,在最后一行添加:

export NDK_HOME=//todo
export PATH=$PATH:$NDK_HOME

注意:NDK_HOME后面的路径就是你解压完成后的绝对路径;

测试

编辑完成后,为了使刚刚修改的文件立即生效,输入以下命令:source ~/.bashrc.最后输入ndk-build命令,如果没有出现这样的提示,就证明安装成功;

ndk-build: command not found

通常情况下,应该显示类似下面的提示:

todo
