科学上网_SS建站笔记

Posted on 2017-05-04 | In Original , SS | Visitors

前言

外面的世界是多么的美丽,但是奈何有长城的存在,挡住了围城中的我们和围城外的他人.
钱钟书的<<围城>>一书中说过这样一句话:城里的人想出去,城外的人想进来.对此,写上一篇出城的方法.
出城的方式主要借用一款开源神器Shadowsocks(以下简称SS,你懂得);

准备工作

  1. 钱(带有Visa标志的信用卡或国内支付宝)
  2. 下载SS客户端(笔者是win的)

购买服务器

奈何笔者没有Visa信用卡,所以用的是搬瓦工的服务器,架设在美国洛杉矶.
查看文档1,这里详细说明了订购服务器的方法,我们按照步骤一步一步的来就可以了.
在按照文档1中的步骤做完后,那么我们订购的服务也是买好了.
注意:

  1. 这里笔者推荐不熟悉的用户先订购一套较为便宜的价格方案,时间订购一个月.在以后熟练后,在可以考虑升级价格方案和订购时间;
  2. 搬瓦工服务器被墙后,我们在订购服务器的时,需翻墙去访问.这里推荐lantern.(先借个梯子,再搭我们的梯子.软件可以从github上下载到,下载安装即可运行)

登录服务器

在我们购买好后,登录我们买好服务器的官网,同样也是采用我们刚刚借的梯子去登录.
登录之后查看文档3,介绍的十分清楚了,不多赘述.
注意:

  1. 在配置过程中遇到Shadowsocks配置失败的情况,如果出现以下错误

    DEPRECATION: Python 2.6 is no longer supported by the Python core team, please upgrade your Python. A future version of pip will drop support for Python 2.6
    这里建议将命令pip instatll shadowsocks换成easy_install shadowsocks即可

下载SS

点我去下载.
在下载好的SS客户端并安装.这里我们开始配置SS的服务器.
右键小飞机 -> 服务器 -> 编辑服务器
出现下图.按照图示配置即可;

打完收工

退出我们借的梯子(lantern),开启我们的SS客户端.
右键 -> 启用系统代理;在代理模式中我们选择PAC模式
打开Google和Youtube分别测试一下;

PAC模式和全局模式

区别

全局模式顾名思义就是 我们整个电脑的网络都走代理.(不推荐,因为这样在访问国内的网站时,也会走我们的服务器流量,造成不必要的消耗,同时网速也变慢)
PAC模式,就是根据我我们配置的文件,选择性的访问网络.(该走代理的走代理,该直接访问的就直接访问)
